Getting Started

What is ACB Link Desktop

ACB Link Desktop is a free application that lets you:

  • Listen to 10 live ACB Media streams
  • Browse and download over 35 podcasts
  • Find ACB state affiliates and special interest groups
  • Record streams and schedule future recordings
  • Create playlists of your favorite content
  • Use voice commands for hands-free control

The application is designed from the ground up to work with screen readers and assistive technology.

System Requirements

Windows Requirements: - Windows 10 or Windows 11 - Both 64-bit Intel and ARM64 processors are supported - 256 megabytes of RAM minimum - 100 megabytes of storage plus space for downloads - Internet connection for streaming

macOS Requirements: - macOS 10.14 Mojave or later - Both Apple Silicon and Intel Macs are supported - 256 megabytes of RAM minimum - 100 megabytes of storage plus space for downloads - Internet connection for streaming

Installing on Windows

  1. Download the installer file from the GitHub Releases page
  2. The file is named ACBLink-Setup.exe
  3. Run the installer file
  4. Follow the prompts in the installation wizard
  5. When installation completes, launch ACB Link from the Start Menu

You can also find a shortcut on your desktop if you chose to create one during installation.

Installing on macOS

  1. Download the DMG file from the GitHub Releases page
  2. The file is named ACBLink-macOS.dmg
  3. Open the DMG file
  4. Drag the ACB Link Desktop icon to your Applications folder
  5. Launch from Applications or use Spotlight to search for ACB Link

The first time you open the app, macOS may ask you to confirm that you want to open software from the internet.

First Launch

When you first open ACB Link Desktop:

  1. The main window appears with the Home tab selected
  2. A local web server starts automatically in the background
  3. The status bar at the bottom shows Ready when everything is loaded
  4. Voice control is available immediately, just say Hey ACB Link to activate

Main Interface

Application Window Layout

The ACB Link window has these main areas, from top to bottom:

  1. Menu Bar: Contains all application menus. Press Alt to access.
  2. Tab Panel: Nine tabs for different content areas
  3. Content Area: Shows the selected tab content
  4. Player Controls: Play, pause, stop, volume, and recording buttons
  5. Status Bar: Shows what is playing and your online status

The Nine Tabs

The application has nine tabs that you can navigate between:

  1. Home: Welcome screen with quick actions
  2. Streams: Live ACB Media streams
  3. Podcasts: Browse all available podcasts
  4. Affiliates: State affiliates and special interest groups
  5. Resources: ACB publications and links
  6. Favorites: Your bookmarked content
  7. Playlists: Custom playlists you have created
  8. Search: Find content across all categories
  9. Calendar: ACB events and scheduled recordings

Tab Navigation

Use these keyboard shortcuts to switch tabs quickly:

  • Control plus Tab: Go to next tab
  • Control plus Shift plus Tab: Go to previous tab
  • Control plus 1: Go to Home tab
  • Control plus 2: Go to Streams tab
  • Control plus 3: Go to Podcasts tab
  • Control plus 4: Go to Affiliates tab
  • Control plus 5: Go to Resources tab
  • Control plus 6: Go to Favorites tab
  • Control plus 7: Go to Playlists tab
  • Control plus 8: Go to Search tab
  • Control plus 9: Go to Calendar tab

Pane Navigation

Use F6 to move between the four main areas of the window:

  1. Tab Bar: The row of tabs at the top
  2. Tab Content: The main content area for the current tab
  3. Player Controls: Playback buttons and volume at the bottom
  4. Status Bar: Status information at the very bottom

Press F6 to move to the next pane, or Shift plus F6 to move to the previous pane. Your screen reader will announce which pane has focus.

Customizing the Home Page

The Home page is your personalized dashboard. You can customize which widgets appear and in what order.

Available Widgets

The Home page can display these widgets:

  • Welcome: Personalized greeting and getting started tips
  • Favorite Streams: Quick access to your bookmarked live streams
  • Favorite Podcasts: Your favorite podcasts with recent episodes
  • Calendar Events: Upcoming ACB events and your scheduled recordings
  • Favorite Affiliates: Quick links to your bookmarked affiliates
  • Quick Actions: Common tasks like play random stream or browse new content
  • Now Playing: Current playback status and controls
  • Recent Episodes: Recently played podcast episodes
  • Download Progress: Active downloads and their status
  • Listening Stats: Your listening history and statistics

Opening the Customize Dialog

  1. Open the View menu with Alt plus V
  2. Select “Customize Home Page…”
  3. The customize dialog opens

Enabling and Disabling Widgets

In the customize dialog:

  1. A checklist shows all available widgets
  2. Check widgets you want to display
  3. Uncheck widgets you want to hide
  4. Press OK to apply changes

Reordering Widgets

To change the order widgets appear on your home page:

  1. Select a widget in the list
  2. Click Move Up to move it higher
  3. Click Move Down to move it lower
  4. The top widget appears first on your home page

Resetting to Defaults

Click the Reset to Defaults button to restore the original widget configuration.

Widget Accessibility Features

Each widget section has:

  • Proper heading levels: Use heading navigation (H key in screen readers) to jump between widgets
  • Collapse/Expand: Double-click or press Enter on a widget heading to collapse or expand it
  • Keyboard navigation: Tab through widget contents, use arrow keys in lists
  • Live regions: Status changes are announced automatically

Tips for Best Experience

  • Enable only the widgets you frequently use
  • Put your most important widgets at the top
  • Collapse widgets you need occasionally but don’t want taking up space
  • The Welcome widget is great for new users but can be hidden once familiar
  • Keep Now Playing enabled if you often use playback controls

Listening to Streams

Available Live Streams

ACB Link provides access to 10 live streaming channels. Each channel has a keyboard shortcut for instant playback.

Stream 1, ACB Media 1: Flagship programming including news, magazines, and special events. Keyboard shortcut Alt plus 1.

Stream 2, ACB Media 2: Additional flagship programming and simulcasts. Keyboard shortcut Alt plus 2.

Stream 3, Old Time Radio: Classic radio dramas, comedies, and variety shows. Keyboard shortcut Alt plus 3.

Stream 4, Advocacy and News: Blindness-related news and advocacy content. Keyboard shortcut Alt plus 4.

Stream 5, Audio Description: Movies and TV shows with audio description. Keyboard shortcut Alt plus 5.

Stream 6, International: Content from ACB international partners. Keyboard shortcut Alt plus 6.

Stream 7, ACB Cafe: Talk shows, interviews, and community content. Keyboard shortcut Alt plus 7.

Stream 8, Classical Music: Classical music performances and concerts. Keyboard shortcut Alt plus 8.

Stream 9, Jazz: Jazz music from various eras and styles. Keyboard shortcut Alt plus 9.

Stream 10, Country: Country music programming. Keyboard shortcut Alt plus 0.

Playing a Stream

Fastest Method using Keyboard Shortcuts: Press Alt plus a number from 1 to 0 to instantly play any stream. For example, Alt plus 3 plays Old Time Radio.

Using the Streams Tab: 1. Press Control plus 2 to go to the Streams tab 2. Use the Up and Down arrow keys to select a stream 3. Press Enter to start playing

Using Voice Control: 1. Say Hey ACB Link and wait for the listening indicator 2. Say Play stream followed by the number, for example, Play stream 5

Playback Controls

While a stream is playing, use these controls:

  • Space bar: Pause or resume playback
  • Control plus Shift plus S: Stop playback completely
  • Control plus Up Arrow: Increase volume
  • Control plus Down Arrow: Decrease volume
  • Control plus M: Toggle mute on and off

Browsing Podcasts

Podcast Library

ACB Link includes over 35 podcasts covering various topics:

  • ACB News and Reports
  • Advocacy and Focus Topics
  • Technology Tips
  • Audio Description and Entertainment
  • Health and Wellness
  • Convention and Community Events
  • Spanish Language Content (Podcasts en español)
  • Many more

Finding Podcasts

  1. Press Control plus 3 to go to the Podcasts tab
  2. The left panel shows all available podcasts
  3. Use the search box to filter podcasts by name
  4. Select a podcast and press Enter to load its episodes
  5. Episodes appear in the right panel
  6. Select an episode and press Enter to stream

Sorting Episodes

Change how episodes are sorted using the View menu:

  1. Open the View menu
  2. Select “Sort Podcast Episodes”
  3. Choose your preferred sort order:
    • Newest First (default)
    • Oldest First
    • Title A-Z
    • Title Z-A
    • Longest First
    • Shortest First
    • Unplayed First
    • Downloaded First

You can also use the sort dropdown directly in the Podcasts panel.

Episode Status

Each episode shows its current status:

  • New: Episode not yet played
  • In Progress: Partially played (will resume from position)
  • Played: Already listened to
  • Downloaded: Available offline

Podcast Playback Controls

In addition to standard playback controls, podcasts support:

  • Control plus Left Arrow: Skip backward 15 seconds
  • Control plus Right Arrow: Skip forward 30 seconds
  • Control plus Shift plus Left: Previous episode
  • Control plus Shift plus Right: Next episode

Streaming vs Downloading

Stream Episode: Play immediately without downloading. Requires internet connection but saves storage space.

Download Episode: Save to your computer for offline listening. Great for travel or unreliable connections.

Downloading Episodes

To download a podcast episode for offline listening:

  1. Select the episode in the list
  2. Click the Download button, or
  3. Press the Applications key and choose Download Episode
  4. The episode saves to your Podcasts folder
  5. Downloaded episodes show “Downloaded” status

To delete a download and reclaim space: 1. Select the downloaded episode 2. Click Delete Download (the button changes when episode is downloaded) 3. The episode remains available for streaming

Episode Context Menu

Right-click on any episode for quick actions:

  • Stream Episode
  • Download Episode / Delete Download
  • Mark as Played / Mark as Unplayed
  • Add Episode to Favorites
  • Copy Episode URL
  • Episode Details (full description and metadata)

ACB Affiliates

Connect with state affiliate organizations and special interest groups from within ACB Link.

Navigating Affiliates

  1. Press Control plus 4 to go to the Affiliates tab
  2. Choose between State Affiliates and Special Interest Groups tabs
  3. Navigate the list with arrow keys
  4. Press Enter to visit the selected organization’s website

Available Information

For each affiliate, you can view: - Organization name - State (for state affiliates) - Contact person name - Email address - Phone number - Website URL - X (Twitter) handle - Facebook page

Actions for Affiliates

Select an affiliate and use these options:

Visit Website: Press Enter or click the Visit Website button to open the organization’s website in your browser.

Send Email: Click the Send Email button to open your email client with the contact’s email address.

Copy Information: Click Copy Info to copy all affiliate details to your clipboard for sharing.

Context Menu: Right-click or press the Applications key for more options: - Visit Website - Send Email - Copy All Info - Copy Website URL - Copy Email Address - Copy Phone Number - Open X (Twitter) - Open Facebook - Suggest Correction

Suggesting Corrections

If you notice outdated or incorrect affiliate information, you can submit a correction request:

  1. Select the affiliate in the list
  2. Right-click and choose Suggest Correction, or click the Suggest Correction button
  3. Alternatively, press Control plus Shift plus E from anywhere in the application
  4. Check the fields you want to correct
  5. Enter the new/correct values
  6. Optionally indicate if you are affiliated with the organization
  7. Submit via GitHub Issues or copy to clipboard for email

Your correction requests help keep the ACB affiliate directory accurate and up-to-date.

To add from a list: 1. Select the item 2. Press the Applications key for the context menu 3. Choose Add to Favorites

Creating Playlists

What are Playlists

Playlists let you organize content into custom collections. You can mix streams, podcast episodes, and recordings in any order.

Creating a New Playlist

  1. Press Control plus 7 to go to the Playlists tab
  2. Press Control plus N for new playlist
  3. Enter a name for your playlist
  4. Press Enter to create it

Adding Items to a Playlist

  1. Navigate to the content you want to add
  2. Press the Applications key for the context menu
  3. Choose Add to Playlist
  4. Select which playlist from the submenu

Playing a Playlist

  1. Go to the Playlists tab
  2. Select your playlist
  3. Press Enter to start playing from the beginning

The playlist plays items in order. Use Control plus Shift plus Right to skip to the next item.

Recording Audio

Recording the Current Stream

To record what you are currently listening to:

  1. Press Control plus R to start recording
  2. The status bar shows Recording in Progress
  3. Press Control plus R again to stop recording

Recording Settings

Before recording, you can configure:

  • Audio Format: MP3, WAV, or OGG
  • Bitrate: 64, 128, 192, or 320 kilobits per second
  • Save Location: Default is your Documents folder

Access these in Settings, then Recording.

Finding Your Recordings

Recordings are saved to your Documents folder by default, in a subfolder called ACB Link Recordings. The filename includes the stream name and timestamp.

Scheduled Tasks

About Scheduled Tasks

ACB Link can schedule recordings to happen automatically, even when the application is not running. This uses:

  • Windows: Task Scheduler
  • macOS: launchd service

Scheduling a Future Recording

  1. Go to the Calendar tab with Control plus 9
  2. Find the event you want to record
  3. Press the Applications key for the context menu
  4. Choose Schedule Recording
  5. Configure the recording options
  6. Press Enter to confirm

The recording will start automatically at the scheduled time.

Viewing Scheduled Tasks

  1. Go to the Tools menu
  2. Choose Scheduled Tasks
  3. View all your pending scheduled recordings

From this dialog you can enable, disable, or delete scheduled tasks.

Automatic Podcast Sync

You can schedule automatic podcast updates:

  1. Go to Settings
  2. Choose the Playback tab
  3. Enable Auto-sync podcast feeds
  4. Set how often to check for new episodes

Data Synchronization

About Data Sync

ACB Link keeps your data current by synchronizing with ACB servers. This includes:

  • Stream information and URLs
  • Podcast feeds and episodes
  • State affiliate information
  • Special interest group listings
  • ACB publications and resources

Automatic Sync

By default, ACB Link checks for data updates when you launch the application. This ensures you always have the latest content.

Manual Sync

To manually synchronize data:

  1. Go to the Tools menu
  2. Choose Sync Data, or press Control plus Shift plus S
  3. A progress dialog shows the sync status
  4. Click OK when complete

Offline Data

ACB Link includes a complete offline copy of all data. If you cannot connect to the internet, the application uses this local data instead.

Sync Settings

Configure synchronization in Settings under the Data tab:

  • Enable or disable automatic sync
  • Set sync frequency
  • Choose which data types to sync
  • View last sync time for each data source

Voice Control

Activating Voice Control

Keyboard Shortcut Method (Default): Press Ctrl+Shift+V to toggle voice control, then speak your command. This method works immediately without any additional setup.

Wake Word Method (Optional - Requires Additional Download): Say your configured wake word (default: "Hey Link") and wait for the listening indicator sound. Then speak your command.

Note: Wake word detection requires optional AI/ML components (~5GB) that are not included in the standard installer. See "Installing Wake Word Support" below for setup instructions.

Continuous Mode: If you disable the wake word requirement in Settings, you can speak commands continuously without saying the wake word first.

Installing Wake Word Support (Optional)wake word feature allows hands-free activation by saying a phrase like "Hey ACB Link". This requires additional AI/ML libraries that are downloaded separately to keep the main installer small.*To install wake word support:**. Open Command Prompt (Windows) or Terminal (macOS). Run: pip install openwakeword torch torchaudio. Wait for download to complete (~5GB, may take several minutes). Restart ACB Link Desktop. Go to Settings > Voice Control to configure your wake word*Without wake word support**, voice control still works fully:- Press Ctrl+Shift+V to activate voice listening- Speak your command- All voice commands function normallyonly difference is you activate by keyboard instead of by speaking a wake phrase.

Voice Commands

Playback Commands: - Play: Resume playback - Pause: Pause playback - Stop: Stop playback completely - Volume up: Increase volume - Volume down: Decrease volume - Mute: Mute audio - Unmute: Restore audio - Skip forward: Skip ahead in current content - Skip back: Skip backward in current content

Stream Commands: - Play stream one through Play stream ten - Play channel one through Play channel ten

Navigation Commands: - Go to home or Go home - Go to streams or Streams tab - Go to podcasts or Podcasts tab - Go to affiliates or Affiliates tab - Go to resources or Resources tab

Recording Commands: - Start recording or Record - Stop recording or End recording

System Commands: - Settings or Open settings: Open settings dialog - What is playing or Now playing: Announces current content - Help or List commands: Lists available commands - Stop listening or Voice off: Deactivate voice control

Voice Control Settings

Configure voice control in Settings under the Voice Control tab:

Wake Word Settings: - Enable or disable wake word requirement - Set custom wake word (e.g., “hey link”, “ok computer”, “hey radio”) - Wake word times out after 10 seconds of silence

Voice Feedback: - Enable voice feedback for spoken confirmations - Enable command confirmation announcements - Enable text-to-speech responses

Text-to-Speech: - Adjust speaking rate from 100 to 300 words per minute - Set TTS volume independently from playback - Choose from available system voices

Key Feedback Sounds: - Enable or disable audio feedback sounds during voice recognition - A “key down” sound plays when voice recognition starts - A “key up” sound plays when voice recognition stops - Use the default sounds or specify custom sound files (MP3, WAV, or OGG) - Custom sound paths can be entered directly or selected via the Browse button

Customizing Voice Commands

You can customize the trigger phrases for any voice command:

  1. Go to Settings, Voice Control tab
  2. Click the Customize Voice Commands button
  3. Select a command from the list on the left
  4. Edit the trigger phrases in the text area (one per line)
  5. Click Save Triggers to save your changes
  6. Click Done when finished

This allows you to: - Add phrases in your preferred style - Use shorter or longer commands - Add regional language variations - Remove triggers you accidentally speak

Customizing Key Feedback Sounds

To use your own audio feedback sounds during voice recognition:

  1. Go to Settings, Voice Control tab
  2. Scroll to the Key Feedback Sounds section
  3. Check “Enable key feedback sounds” to turn sounds on or off
  4. For custom sounds, click Browse next to “Key down sound” or “Key up sound”
  5. Select an MP3, WAV, or OGG audio file
  6. Leave the paths empty to use the default sounds

The default sounds are located in the sounds folder (key_down.mp3 and key_up.mp3).

Quick Actions

Quick Status (F5)

Press F5 at any time to hear a spoken announcement of the current playback status, including what is playing, the playback state, and the volume level. This is especially helpful for screen reader users who want to quickly check the current state without navigating through the interface.

Panic Key (Control plus Shift plus M)

The panic key instantly mutes audio and minimizes the application. This is useful when you need to quickly silence the audio, such as when receiving a phone call or when someone enters the room. Press the keyboard shortcut Control plus Shift plus M to activate.

Text Size Adjustment

Quickly adjust the text size without opening Settings:

  • Control plus Plus: Make text larger
  • Control plus Minus: Make text smaller
  • Control plus 0: Reset to default size

Changes are saved automatically and apply immediately throughout the application.

Resume Playback

When you start ACB Link, if you were previously listening to a stream or podcast, you will be asked if you want to resume where you left off. Choose Resume to continue, or Start Fresh to begin normally.

Playback Enhancements

Sleep Timer

The sleep timer lets you set ACB Link to stop playback after a specified time, perfect for listening in bed.

To set a sleep timer: 1. Press Control plus T or go to Playback, then Sleep Timer 2. Choose a preset time: 15, 30, 45, 60, 90, or 120 minutes 3. Or enter a custom time in minutes 4. Optionally enable volume fade-out for a gentle transition 5. Click a preset button or Set to start the timer

When the timer is active, the remaining time shows in the status bar. To cancel, open the Sleep Timer dialog and click Cancel Timer.

Playback Speed

Control how fast or slow audio plays back, useful for podcasts and long-form content.

To adjust playback speed: 1. Press Control plus P or go to Playback, then Playback Speed 2. Choose a preset: 0.5x, 0.75x, 1x, 1.25x, 1.5x, 1.75x, or 2x 3. Or use the slider for fine control 4. Click Reset to Normal to return to standard speed

Your speed preference is saved and applied to future playback.

Copy What’s Playing

Share what you are listening to by copying the current track information to your clipboard.

Press Control plus Shift plus C or go to Playback, then Copy What’s Playing. The copied text includes the stream name, current track if available, and a link. Paste it anywhere like an email or social media post.

Recently Played

Quickly access your recently played streams and podcasts.

Press Control plus Shift plus R or go to Playback, then Recently Played. A dialog shows your 20 most recent items. Double-click or press Enter to play a selected item. Use the Clear All button to remove history.

Auto-Play on Startup

Configure ACB Link to automatically start playing when you launch the application.

To enable auto-play: 1. Go to Tools, then Auto-Play on Startup 2. Check Enable auto-play on startup 3. Choose what to play: Last played content or a specific stream 4. Set the delay before auto-play starts (default: 3 seconds) 5. Optionally, only auto-play if your last session was playing 6. Click OK to save

Quiet Hours

Automatically reduce volume or mute notifications during specified times.

To configure quiet hours: 1. Go to Tools, then Quiet Hours 2. Check Enable quiet hours 3. Set start time (example: 22:00 for 10 PM) 4. Set end time (example: 07:00 for 7 AM) 5. Choose options: Mute notifications, Reduce volume 6. Adjust reduced volume level 7. Click OK to save

Quiet hours automatically activate during the specified times every day.

Audio Ducking

Audio ducking temporarily lowers the volume when system sounds play, so you do not miss important notifications.

To toggle audio ducking: Enable audio ducking from Tools, then check Audio Ducking in the menu.

To configure audio ducking settings: 1. Go to Tools, then Audio Ducking Settings 2. Check Enable audio ducking to turn the feature on or off 3. Adjust the Duck Percentage slider (10% to 90%) - this is how much volume reduces during system sounds 4. Adjust the Restore Delay slider (0.5 to 10 seconds) - this is how long before volume returns to normal 5. Click Test Ducking to preview the effect with current settings 6. Click OK to save your settings

Default values are 30% for duck percentage and 2 seconds for restore delay.

Media Key Support

Use your keyboard’s media keys to control playback without focusing the application.

Enable media keys from Tools, then check Media Key Support in the menu. Supported keys include Play/Pause, Stop, and Volume Up/Down on keyboards that have dedicated media buttons.

View Settings and Layout

Customizing the Interface

ACB Link allows you to show or hide various parts of the interface to suit your preferences. All visibility settings are saved and persist between sessions.

Show or Hide UI Elements

From the View menu, you can toggle visibility of:

  • Toolbar: The button bar below the menu
  • Tab Bar: The row of tabs for navigating sections
  • Status Bar: The information bar at the bottom of the window
  • Player Controls: The playback buttons and volume slider

Keyboard Shortcuts for View Toggles

Available panes include the main content area, player controls, and sidebar (when visible). Screen reader users will hear an announcement when focus moves to a new pane.

Focus Mode

Focus Mode provides a minimal, distraction-free interface showing only the most essential controls.

When enabled, Focus Mode hides the toolbar, tab bar, status bar, and sidebar, keeping only the main content and player controls visible. Exit Focus Mode using the same shortcut to restore your previous layout.

Full Screen Mode

Enable accessibility options on the Accessibility tab 5. Click Apply to preview changes, or OK to save and close

Listening Statistics

Export and Import Settings

Backing Up Your Settings

You can create a backup of all your settings, favorites, playlists, and data:

  1. Go to Tools, then Export Import Settings
  2. Click Export Backup
  3. Choose a location to save the backup file
  4. The backup is saved with a dot acbbackup extension

Restoring from Backup

Advanced Settings

Advanced Settings provide access to internal configuration options. Most users will never need to use these settings.

Administrator Access Required

Starting with version 1.1.0, Advanced Settings require administrator authentication. Your admin role is determined by your GitHub permissions on the ACB Link configuration repository.

Who can access Advanced Settings: - CONFIG_ADMIN (GitHub repo admin/maintain permission) - SUPER_ADMIN (GitHub organization owner)

Regular users and Affiliate Admins cannot access Advanced Settings to prevent accidental configuration changes.

Warning

Changing advanced settings incorrectly can cause the application to malfunction or fail to connect to ACB media services. Only make changes if you understand their effects.

Accessing Advanced Settings

  1. Press Control plus Shift plus Comma, or
  2. Go to File menu and select Advanced Settings
  3. If not logged in, you’ll be prompted to enter your GitHub Personal Access Token
  4. Read the warning dialog and click Proceed to continue

Creating a GitHub Token

If you are an administrator and need to access Advanced Settings:

  1. Go to https://github.com/settings/tokens/new
  2. Set description: “ACB Link Desktop Admin”
  3. Select scope: read:org (this is the only scope needed)
  4. Click Generate token
  5. Copy the token (it won’t be shown again)
  6. Enter the token in ACB Link when prompted

Your admin role is automatically determined by your GitHub permissions on the config repository.

Troubleshooting

Stream Does Not Play

If a stream will not play:

  1. Check your internet connection
  2. Verify the stream is currently broadcasting
  3. Try a different stream to test connectivity
  4. Go to Settings and try a different audio backend
  5. Restart ACB Link

No Sound

If you do not hear audio:

  1. Check the volume slider in ACB Link
  2. Press Control plus M to check if mute is active
  3. Check your system volume in the Windows or macOS mixer
  4. Verify the correct audio output device is selected
  5. Try playing audio from another application

Voice Control Not Responding

If voice control does not work:

  1. Check that your microphone is connected
  2. Verify microphone permissions in system settings
  3. Go to Settings and check the microphone selection
  4. Reduce background noise in your environment
  5. Speak clearly at normal volume

Screen Reader Not Announcing

If your screen reader is silent:

  1. Verify the screen reader is running
  2. Check that accessible output is enabled in Settings
  3. Try restarting the screen reader
  4. Try restarting ACB Link

Scheduled Tasks Not Running

Windows: 1. Open Task Scheduler from the Start Menu 2. Look for tasks beginning with ACBLink 3. Check the task history for error messages 4. Verify your user account has permission to run scheduled tasks

macOS: 1. Open Terminal 2. Type: launchctl list and press Enter 3. Look for entries containing acb 4. Check that launch agents are enabled

Suggesting Affiliate Corrections

If you notice incorrect information for a state affiliate or special interest group (SIG), you can suggest corrections:

  1. Press Control Shift E, or go to Help, Suggest Affiliate Correction
  2. Select the affiliate you want to correct from the dropdown
  3. Check the boxes for the fields you want to correct (Organization Name, Contact Person, Email, etc.)
  4. Enter the correct values in the text fields that appear
  5. Add any additional notes about the correction
  6. Choose a submission method:
    • Submit for Review: Saves the correction locally for administrator approval (recommended)

    • Copy to Clipboard: For email submission

The “Submit for Review” option is the recommended method as it allows corrections to be reviewed and automatically applied without manual editing of data files.

Affiliate Correction Administration (Administrators Only)

If you are an administrator (AFFILIATE_ADMIN, CONFIG_ADMIN, or SUPER_ADMIN), you can review and apply corrections submitted by users.

Administrator Authentication:

Starting with version 1.1.0, this feature requires GitHub-based authentication:

  1. When you first access the admin panel, you’ll be prompted for your GitHub Personal Access Token
  2. Create a token at https://github.com/settings/tokens/new with the read:org scope
  3. Your admin role is determined by your GitHub permissions on the ACB Link config repository:
    • Repository Write permission → AFFILIATE_ADMIN
    • Repository Admin permission → CONFIG_ADMIN
    • Organization Owner → SUPER_ADMIN

Using the Admin Panel:

  1. Press Control Shift A, or go to Tools, Affiliate Correction Admin
  2. If not logged in, enter your GitHub token when prompted
  3. The review panel shows pending corrections by default
  4. Use the filter dropdown to view Pending, Approved, Rejected, Applied, or All corrections
  5. Select a correction from the list to see basic details
  6. Click View Details to see the full correction information
  7. For pending corrections:
    • Click Approve to accept the correction
    • Click Reject to decline it (you will be asked for a reason)
  8. For approved corrections:
    • Click Apply to Data to update the XML file with the correction
    • Or click Apply All Approved to process all approved corrections at once

Important notes for administrators: - A backup is automatically created before any changes are made - The audit log tracks all approvals, rejections, and applications - If something goes wrong, backups are stored in the data backups folder - Your GitHub username is recorded in the audit log for accountability - You can log out at any time from the session panel - Always verify corrections before approving to maintain data accuracy
